# bmad-vnpt-web-vue

BMAD custom package for modern Vue 3 web applications.

## Design defaults
- Vue 3 + Vite + TypeScript
- Composition API with `<script setup>`
- Vue Router for client-side routing
- Pinia for shared application state
- Vitest + Vue Test Utils for tests
- code-split routes and composable-first business logic

## Package contents
- OpenCode command: `bmad-vnpt-web-vue`
- OpenCode skill: `bmad-vnpt-web-vue`
- Workflow: `bmad-vnpt-web-vue`
- Skeleton starter structure
- Helper scripts for verify / scaffold guidance
