# bmad-vnpt-web-angular

BMAD custom package for Angular web application work.

Default stance:
- Standalone-first Angular architecture
- TypeScript strict-friendly code
- Signals-first for local UI/view state
- RxJS + HttpClient for async/server flows, with `toSignal` / `toObservable` interop where it genuinely helps
- Lazy-loaded routes and deferrable views for performance-sensitive features
- Feature-oriented foldering with thin components and richer data/state services
- Angular style guide alignment
- Testing-first with Vitest default expectations for modern Angular work

Package contents:
- OpenCode command: `bmad-vnpt-web-angular`
- Skill: `.opencode/skills/bmad-vnpt-web-angular`
- Workflow registration under `src/workflows/`
- Templates, scripts, and a standalone Angular skeleton
