# bmad-vnpt-dotnet

BMAD custom for modern .NET / ASP.NET Core work.

Focus:
- ASP.NET Core API-first development
- Minimal APIs by default for new HTTP APIs
- Clean boundaries: Domain / Application / Infrastructure / Api
- EF Core performance-aware access patterns
- Strong validation, observability, health checks, rate limiting, and testability
- xUnit-first testing with integration coverage for real request flows
