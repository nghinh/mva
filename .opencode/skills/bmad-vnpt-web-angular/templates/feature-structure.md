# Recommended Angular Feature Structure

```text
src/app/features/<feature>/
  data/
    <feature>.api.ts
  models/
    <feature>.model.ts
  pages/
    <feature>-page.component.ts
    <feature>-page.component.html
    <feature>-page.component.css
  state/
    <feature>.store.ts
  <feature>.routes.ts
```
